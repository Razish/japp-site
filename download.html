Installation Instructions

Ideally, JA++ is installed without having JA+ already installed.
If JA+ is installed, you will have to remove it to avoid conflicts

If you already have JA+ installed

  1. Remove the JA+ GLA animations from your GameData/base folder
  2. Remove clientPlugin_v*.pk3 from GameData/japlus
  3. Place the downloaded files in GameData/japlus
  4. Start the game with +set fs_game "japlus" as your command line arguments (e.g. .bat file, shortcut, with UU)

If you don't have JA+ installed

  1. Place the downloaded files in GameData/japlus
  2. Start the game with +set fs_game "japlus" as your command line arguments (e.g. .bat file, shortcut, with UU)

Download Links

JA++ is available for Windows, Mac and Linux
Due to not owning a Mac, the Mac builds will lag behind, and are provided by redsaurus in his spare time

You will have to select the correct files for your operating system and CPU architecture (i.e. 32 bit, 64 bit)

JA++ runs best on OpenJK
